Project Overview

Supplier Bridge is a website for import/export businesses that connects clients with suppliers. The project combines a professional website (frontend) with Python programs (backend) for managing inquiries and matching suppliers.

Technologies Used:
HTML, CSS, JavaScript (frontend)
Python (backend)
Project Files
index.html
about.html
services.html
blog.html
contact.html
style.css
script.js
contact_manager.py
supplier_match.py
inquiries.txt
README.md

File Descriptions:
index.html – Homepage with intro and navigation
about.html – About the company and mission
services.html – Business services offered
blog.html – Business tips and supplier info
contact.html – Contact form for clients
style.css – Website design and layout
script.js – Basic website interactivity
contact_manager.py – Collects and saves client inquiries in inquiries.txt
supplier_match.py – Suggests supplier country based on product

Features:
Multi-page website with navigation
Contact system for clients
Supplier inquiry management (Python)
Product-supplier matching (Python)
Data stored in a text file

How to Run:
Open .html files in a browser
Run Python files in a Python environment and follow prompts

Conclusion:
This project demonstrates web development and Python programming together. The website provides business information, while Python handles inquiries and supplier matching.
